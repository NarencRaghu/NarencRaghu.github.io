What is this Project?

While interning at Arizona State University in the Health Systems Engineering Department, I am pursuing a project where I analyzed the correlation between MRI and molecular imaging. How are structure and function can be correlated in the adult mouse brain while keeping in mind the larger context of gene expression?

A Little about the Method:

Using an adult mouse brain, we ran multiple correlation studies between MRI data and molecular information, one of them being a Pearson correlation. Initial stpes included building a SQL database and registering gene expression values. The project is still ongoing and we are currently visualizing the gene expression using Brainrender software in python. Some images can be seen on this page.
