What is this Project?

There are various treatment options for those diagnosed with cancer. Choosing the best option is an important choice for each patient, especially when considering factors such as money and effectivenes of the treatment. This project aims to bring personalized cancer medicine to each patient by connecting each patient with the best choice of treatment for them.

How it Works:

The algorithms use various machine learning algorithms and computational tools to device a model prediction for how each treatment option will impact the status of the patient. This is an ongoing project and this page is still being built.

The Different Options

  • - Surgery
  • - Chemotherapy
  • - Radiation therapy
  • - Bone marrow transplant
  • - Immunotherapy
  • - Hormone therapy
  • - Targeted drug therapy
  • - ryoablation
